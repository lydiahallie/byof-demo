# Demo

### About

Custom demo framework that uses the Build Output API to support:

- Static Rendering
- SSR
- Edge Rendering
- Automatic Image Optimization

### How to use

- Installing: `yarn install`
- Building: `yarn build:css && yarn build`
- Deploying: `yarn deploy`
